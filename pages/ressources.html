QGIS est un système d'information géographique (SIG) complet, convivial et libre qui fonctionne sur les plates-formes Unix, Windows et MacOS.

  • Lien vers le téléchargement de QGIS pour votre plate-forme : Download · QGIS Web Site

Le logiciel Graphab permet de modéliser les réseaux écologiques à partir de la théorie des graphes. Il est facilement interfacé avec les Systèmes d’Information Géographique. Pour faciliter sa prise en main une extension QGIS a été développée. Elle est disponible sur le dépôt officiel de QGIS.

  • Graphab nécessite Java pour fonctionner. Il suffit de télécharger et installer Java sur https://adoptium.net/. Graphab 3.0 nécessite au moins la version 11 de Java.
  • Lien vers la présentation de Graphab : Graphab

MitiConnect est une extension QGIS qui permet d'évaluer les continuités écologiques lors de la séquence ERC (Eviter-Réduire-Compenser). C'est un outil tout-en-un avec une interface graphique dédiée qui permet à l'utilisateur de définir les espèces cibles, les coefficients de friction, les scénarios de la séquence, etc., puis fait appel à QGIS et Graphab pour calculer le résultat de chaque étape.
